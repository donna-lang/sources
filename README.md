# Donna Bootstrap Sources

<img width="180" align="right" alt="Donna bootstrap" src="assets/donna-bootstrap.png" />

This repository contains generated bootstrap sources for the Donna compiler.

The files here are used to build Donna from source before a local Donna compiler
already exists. They are produced from the compiler release pipeline for the
supported QBE targets.

## Generated Repository

⚠️  Do not edit this repository by hand.

All source files, manifests, checksums, and target artifacts in this repository
are generated automatically from the Donna compiler repository. Manual changes
will be overwritten by the next bootstrap update.

To contribute to Donna visit the compiler repository instead:

https://github.com/donna-lang/donna
